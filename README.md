# Junctek-KM-F-Series-BLE-scanner-EspHome
BLE scanner EspHome for Junctek KM105F KM110F KM140F KM160F KM-F Series.

The scanner is designed to obtain information from battery monitors such as KM105F KM110F KM140F KM160F. Tested on KM140F.

It can also work on other series of battery monitors from JUNCTEK.
